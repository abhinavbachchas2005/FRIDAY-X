# FRIDAY X

FRIDAY X is a next-generation AI desktop assistant inspired by futuristic AI systems.

## Vision

Build a personal AI assistant that can:

- 🎙️ Understand voice commands
- 🧠 Remember conversations
- 💻 Control the computer
- 🌐 Browse the web
- 👀 Understand the screen
- 🤖 Use multiple AI agents
- 🔒 Respond only to the owner's voice
- 🎨 Provide a beautiful futuristic interface
